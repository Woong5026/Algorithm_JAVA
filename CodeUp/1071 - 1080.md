### 1071

입력된 정수를 줄을 바꿔 하나씩 출력하는데, 0이 입력되면 종료한다.

```java

import java.util.Scanner;

public class Main {
    public static void main(String[] args){

        Scanner sc = new Scanner(System.in);

        while (true){
            int a = sc.nextInt();

            if(a == 0){
                break;

            }
            System.out.println(a);
        }
    }
}

```


### 1072

n개의 정수를 한 개씩 줄을 바꿔 출력한다.

```java

import java.util.Scanner;

public class Main {
    public static void main(String[] args){

        Scanner sc = new Scanner(System.in);

        int a = sc.nextInt();
        int b[] = new int[a];

        for(int i = 0; i < a; i++){
            b[i] = sc.nextInt();
        }
        for(int i = 0; i < a; i++){
            System.out.println(b[i]);
        }
    }
}

```

### 1074

1씩 줄이면서 한 줄에 하나씩 1이 될 때까지 출력한다.

```java

import java.util.Scanner;

public class Main {
    public static void main(String[] args){

        Scanner sc = new Scanner(System.in);

        int a = sc.nextInt();

        while (true){

            if(a == 0){
                break;
            }
            System.out.println(a);
            a--;
        }
    }
}

```

### 1075

1씩 줄이면서 한 줄에 하나씩 0이 될 때까지 출력한다.

```java

import java.util.Scanner;

public class Main {
    public static void main(String[] args){

        Scanner sc = new Scanner(System.in);

        int a = sc.nextInt();

        while (a < 101 && a >= 1){

            if(a == 0){
                break;
            }
            System.out.println(a - 1);
            a--;
        }
    }
}

```
